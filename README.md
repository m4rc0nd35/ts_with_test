# Awesome Project Build with TypeORM

Steps to run this project:

1. Run `npm i` command
2. Setup database settings inside `ormconfig.json` file
3. Run `npm start` command


Commands npm/yarn:
```
	dev
	typeorm:run
	typeorm:revert
	typeorm:schema:sync
	typeorm:schema:drop
	typeorm:entity:create
	typeorm:cache:clear
	test
 ```
